# SVE-Dataset
The Dataset contains six differet soccer event classes including:

1. Goal
2. Head Goal
3. Penalty Goal
4. Penalty Save
5. Red Card
6. Substitute

Here is the link to access our newly created Soccer Videos Dataset.
